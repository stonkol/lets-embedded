# learn-embedded

## How to write the learning folders

1. `$course` = Course short name
1. `$teacher` = teacher initials (A.F.)
1. `$source` = ud(udemy), edx, yt(Youtube), another website....

Folder name = `$course`-`$teacher`-`$source`
